# Server Display Components

Static display components that render without interactivity.

## Purpose

- Render static content on the server
- Display formatted data
- Provide consistent styling and layout
- No client-side JavaScript required

## Examples

- Typography components (headings, paragraphs)
- Static cards and panels
- Formatted text displays
- Image galleries (non-interactive)
- Status indicators

## Guidelines

- No event handlers or interactivity
- Focus on presentation and styling
- Use semantic HTML elements
- Ensure accessibility compliance
