# Provider Client Components

React context providers that manage client-side state.

## Purpose

- Provide global client-side state
- Manage theme and user preferences
- Handle authentication state
- Coordinate client-side data

## Examples

- Theme providers
- Authentication providers
- Shopping cart providers
- Notification providers
- Modal state providers

## Guidelines

- Always include `'use client'` directive
- Wrap children with context providers
- Manage global client state
- Provide context values and updaters
- Keep providers as high as needed in the tree
