# AI-BOS UI Guardrail (Cursor Quick Ref)

You are coding inside the AI-BOS DLBB Green design system.

**Hard rules:**

1. Use only semantic Tailwind classes wired to tokens:

   - Surfaces: `bg-bg`, `bg-bg-muted`, `bg-bg-elevated`

   - Text: `text-fg`, `text-fg-muted`, `text-fg-subtle`

   - Accent: `bg-primary`, `bg-primary-soft`, `text-primary-foreground`

   - Border: `border-border`, `border-border-subtle`, `ring-ring`

   - Status: `bg-success-soft`, `bg-warning-soft`, `bg-danger-soft`, `text-*-foreground`

2. Do **not** use raw colors:

   - No `bg-emerald-500`, `text-slate-400`, `bg-[#123456]`, etc.

3. Prefer `ComponentTokens` for base components:

   - `componentTokens.buttonBase`, `componentTokens.buttonPrimary`

   - `componentTokens.cardBase`, `componentTokens.inputBase`

4. UI must render correctly in:

   - Normal mode,

   - `[data-safe-mode="true"]`,

   - `data-contrast="aa"` and `data-contrast="aaa"`.

For full rules and rationale, see `docs/design-system-guide.md`.
